Customização de Dashboard

Bem-vindo à área de customização do seu dashboard!

Aqui você poderá criar visualizações personalizadas com base em seus dados hospitalares.

Como funciona?

  1. Upload do CSV: Faça o upload do seu arquivo CSV contendo os dados que deseja visualizar.
  2. Configuração de Gráficos: Escolha os tipos de gráficos que deseja usar para cada coluna de dados.
  3. Visualização: Gere e visualize os gráficos de maneira interativa.
  4. Salvar Dashboard: Salve sua configuração personalizada para acessá-la posteriormente.
